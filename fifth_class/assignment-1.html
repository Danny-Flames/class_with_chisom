Variables and Data Types

(1) Create three variables for each of these data types - string, number, float, boolean, object array.
(Meaning you have three string variables, three number variables and so on...)

(2) Create an object variable of any name with five keys, let the value of each key differ from one another.
key-1 can have a value of string datatype, key-2 can have a value of number data type and so on...

(3) Create an array that has a length of 10. Let the array items be of any data type but let the
5th item be a boolean data type, 7th item be a float, 9th item be a string.

(4) Create a string, get the last character(letter) in the string using one of the
string methods made available by javascript, convert it to capital letter, save this converted
character(capital letter) in a variable and output the variable to the console.

(5) Create a string of length more than 24, get the text from the 8th index till the 15th index using
one of javascript's string methods convert the extracted text to capital letter, save this converted
text(capital letter) in a variable and output the variable to the console.
